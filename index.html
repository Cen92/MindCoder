About Mind Coder

Mind Coder is an Android application for programming Lego Mindstorms. You can write some code on an Android device, interpret and evaluate this code using ANTLR and send an output to a Mindstorm v2.0 to run.

The functions are as follows

moveForward(int distance)
moveBackward(int distance)
turnLeft(int degrees)
turnRight(int degrees)
shoot(int numberOfShots)

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.
